# Beacon Board

Beacon Board is a fresh, empty product repository for coordinating launch
checklists. The source harness may be copied from a developer checkout, but
local-only source artifacts such as `.git`, `.DS_Store`, `node_modules`,
`CLAUDE.local.md`, and tool-local settings must not appear in the target
workspace.

The product workflow is known. Runtime, framework, database, package manager,
deployment provider, and test commands have not been chosen yet.
